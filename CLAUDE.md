# Castle Academy

## Design System
Always read `DESIGN.md` before making any visual or UI decisions.
All font choices, colors, spacing, aesthetic direction, and the chess-board
treatments are defined there ("Paper Mono"). It governs every surface — the
marketing site, the student app, and the tutor app.
Do not deviate without explicit user approval.
In QA or code review, flag any UI that doesn't match DESIGN.md.
